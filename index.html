[ ABOUT ]

Cathode Arc is an indie game studio founded by a plasma physicist who traded particle accelerators for pixel art.

We build games the old-fashioned way: vanilla JavaScript, HTML5 Canvas, and the Web Audio API. No frameworks. No build tools. No node_modules folder the size of a black hole. Just clean, handcrafted code that runs in any browser.

Every sprite is drawn procedurally. Every sound is synthesized in real time. Every game ships as two files—an HTML shell and a single JS file—because constraints breed creativity.
